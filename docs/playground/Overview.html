DSC summary & query¶

This page displays contents of database dsc_result.db generated by this DSC. There are two types of tables in the database:

  • pipeline tables: describes how module instances are connected to a pipeline, via module instance ID.
  • module tables: parameters and dependencies of module instances.
In [1]:
import pickle
data = pickle.load(open("dsc_result/dsc_result.db", 'rb'))
